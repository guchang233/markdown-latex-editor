MarkTeX 编辑器启动页

提示: 此页面是为了帮助您在无法直接使用 npm 命令时启动编辑器。

方法一：直接启动（推荐）

点击下面的按钮在新窗口中打开编辑器：

启动 MarkTeX 编辑器

方法二：解决 PowerShell 执行策略限制

如果您无法通过 npm run dev 命令启动应用，请按照以下步骤解决 PowerShell 执行策略限制：

  1. 使用命令提示符 (CMD) 而不是 PowerShell：

    cmd /c "npm run dev"

  2. 或者，以管理员身份打开 PowerShell 并执行以下命令：

    Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

    然后您可以正常使用 npm run dev 命令

方法三：使用修复脚本

运行修复脚本以解决依赖问题：

node fix-project.js

疑难解答:

如果编辑器仍然无法正常工作，请尝试以下步骤：

  • 清除浏览器缓存并重新加载
  • 确保运行了最新版本的修复脚本
  • 检查控制台是否有错误信息（按 F12 打开开发者工具）
